Contiene los archivos que describen los pasos para realizar cotizaciones de la empresa TK-U del proyecto final del curso Introducción a la Programación de Computadoras.
Inicialmente en el archivo proyecto.js se declara un variable llamada opcion = "";
Posteriormente se declara un ciclo while, teniendo de condicion opcion !=== "Salir"
Luego se tiene las áreas de la declaración de las variables usadas por el programa para generar los calculos necesarios.
Funciona de la siguiente manera.
1. Declaración de las variables usadas para los calculos
2. Sentencias de condicionales para validar la información a ser ingresada
3. Cambio del flujo del programa en base a las respuestas del usuario
4. Realiza el calculo de cada sección de acuerdo con las respuestas y valores ingresados por el usuairo
5. Suma de los calculos realizados a la variable subtotal
6. Mostrar los resultados de los calculos y las cotización realizada
Las secciones del funcionamiento a considerar del programa son:
1. Declaración de variable inicial y del ciclo while
2. Declaración de variables de los calculos y sentencias condicionales
3. Área para mostrar los calculos hechos por el programa.
Considero que las preguntas podrían agruparse en secciones de temas específicos, para mostrar la información de una manera más clasifiado y el usuario pueda navegar por esas secciones.
